# TC_PUMB_001: Відображення головної сторінки реєстрації ФОП

## Основна інформація
- **ID:** TC_PUMB_001
- **Title:** Перевірка коректного відображення головної сторінки реєстрації ФОП
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** YES
- **References:** US_PUMB_001

## Preconditions
- Доступ до інтернету
- Браузер відкритий
- URL: https://www.digital.pumb.ua/registration/fop/choose

## Steps
1. Відкрити URL сторінки реєстрації ФОП
2. Дочекатися повного завантаження сторінки
3. Перевірити відображення основного заголовка
4. Перевірити наявність двох варіантів реєстрації

## Expected Result
Сторінка завантажується коректно. Відображається заголовок "ВІДКРИВАЄМО РАХУНОК ДЛЯ ФОП" та підзаголовок "Оберіть спосіб відкриття рахунку". Присутні два варіанти: "Без участі менеджера" та "За участю менеджера".

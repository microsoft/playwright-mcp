# TC_PUMB_005: Перевірка відображення вимог для відеоверифікації

## Основна інформація
- **ID:** TC_PUMB_005
- **Title:** Детальна перевірка вимог для реєстрації з менеджером
- **Type:** Functional
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** YES
- **References:** US_PUMB_005

## Preconditions
- Сторінка реєстрації ФОП завантажена
- Секція "За участю менеджера" відображається

## Steps
1. Знайти секцію "За участю менеджера"
2. Перевірити заголовок "Для відкриття рахунку за допомогою менеджера необхідні:"
3. Перевірити вимогу про камеру
4. Перевірити вимогу про документи
5. Перевірити вимогу про електронний підпис

## Expected Result
Відображається повний список вимог для відеоверифікації. Вимога 1: "Камера на телефоні або комп'ютері для відеозв'язку з працівником банку". Вимога 2: "Скан-копії документів (паспорт, РНОКПП)". Вимога 3: "Кваліфікований електронний підпис". Всі вимоги відображаються у вигляді списку.

## Postconditions
Не потрібно

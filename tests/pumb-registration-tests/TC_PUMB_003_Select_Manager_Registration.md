# TC_PUMB_003: Вибір способу "За участю менеджера"

## Основна інформація
- **ID:** TC_PUMB_003
- **Title:** Перевірка вибору реєстрації з участю менеджера
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 4m
- **Automated:** YES
- **References:** US_PUMB_003

## Preconditions
- Сторінка реєстрації ФОП завантажена
- Відображаються два варіанти реєстрації

## Steps
1. Знайти секцію "За участю менеджера"
2. Перевірити відображення підзаголовка "Відеоверифікація"
3. Перевірити список вимог для відеоверифікації
4. Натиснути на кнопку/посилання для переходу до відеоверифікації

## Expected Result
Відображається секція "За участю менеджера" з підзаголовком "Відеоверифікація". Показані вимоги: "Камера на телефоні або комп'ютері для відеозв'язку з працівником банку", "Скан-копії документів (паспорт, РНОКПП)", "Кваліфікований електронний підпис". Кнопка активна та дозволяє перейти до наступного кроку.

## Postconditions
Повернутися на головну сторінку реєстрації

# TC_PUMB_004: Перевірка відображення вимог для онлайн реєстрації

## Основна інформація
- **ID:** TC_PUMB_004
- **Title:** Детальна перевірка вимог для самостійної онлайн реєстрації
- **Type:** Functional
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** YES
- **References:** US_PUMB_004

## Preconditions
- Сторінка реєстрації ФОП завантажена
- Секція "Без участі менеджера" відображається

## Steps
1. Знайти секцію "Без участі менеджера"
2. Перевірити заголовок "Для самостійного відкриття рахунку онлайн необхідні:"
3. Перевірити першу вимогу про застосунок Дія
4. Перевірити другу вимогу про застосунок ПУМБ
5. Перевірити логічний розділювач "або"

## Expected Result
Відображається повний список вимог для онлайн реєстрації. Перша вимога: "Застосунок Дія з ID-карткою або закордонним паспортом". Друга вимога: "Застосунок ПУМБ для фізичних осіб (Для діючих клієнтів із карткою у застосунку)". Між вимогами присутній розділювач "або".

## Postconditions
Не потрібно

# Тест-сьют: Реєстрація ФОП - ПУМБ Бізнес

## Опис тест-сьюту
Цей тест-сьют покриває функціонал сторінки реєстрації ФОП (фізичної особи-підприємця) на сайті ПУМБ Бізнес, включаючи вибір способу відкриття рахунку, UI тестування та валідацію.

## Загальна статистика
- **Загальна кількість тест-кейсів:** 15
- **Позитивні тест-кейси:** 6
- **Негативні тест-кейси:** 3
- **UI тест-кейси:** 4
- **Тест-кейси валідації:** 2

## Розподіл за пріоритетами
- **Critical:** 0
- **High:** 8
- **Medium:** 5
- **Low:** 2

## Розподіл за типами
- **Functional:** 9
- **UI:** 4
- **Validation:** 2

## Список тест-кейсів

### Позитивні тест-кейси
1. **TC_PUMB_001** - Відображення головної сторінки реєстрації ФОП (High Priority)
2. **TC_PUMB_002** - Вибір способу "Без участі менеджера" (High Priority)
3. **TC_PUMB_003** - Вибір способу "За участю менеджера" (High Priority)
4. **TC_PUMB_004** - Перевірка відображення вимог для онлайн реєстрації (Medium Priority)
5. **TC_PUMB_005** - Перевірка відображення вимог для відеоверифікації (Medium Priority)
6. **TC_PUMB_006** - Клік по кнопці "Повернутись" (Low Priority)

### Негативні тест-кейси
7. **TC_PUMB_007** - Перевірка поведінки при відсутності JavaScript (High Priority)
8. **TC_PUMB_008** - Перевірка поведінки при медленному інтернет-з'єднанні (Medium Priority)
9. **TC_PUMB_009** - Перевірка поведінки при відключенні cookies (Low Priority)

### UI тест-кейси
10. **TC_PUMB_010** - Адаптивність сторінки на різних розмірах екрану (High Priority)
11. **TC_PUMB_011** - Перевірка відображення промо-банера (High Priority)
12. **TC_PUMB_012** - Перевірка відображення контактної інформації (Medium Priority)
13. **TC_PUMB_013** - Перевірка доступності (accessibility) (Medium Priority)

### Тест-кейси валідації
14. **TC_PUMB_014** - Валідація HTML структури сторінки (High Priority)
15. **TC_PUMB_015** - Перевірка мета-тегів та SEO (Medium Priority)

## Загальний час виконання
**Орієнтовний час виконання всього тест-сьюту:** 45 хвилин

## Передумови для виконання тестів
- Доступ до інтернету
- Різні браузери (Chrome, Firefox, Safari)
- Різні розміри екрану для тестування адаптивності
- Інструменти для тестування доступності

## Автоматизація
Всі тест-кейси покриті автотестами на Playwright з можливістю запуску в CI/CD pipeline.

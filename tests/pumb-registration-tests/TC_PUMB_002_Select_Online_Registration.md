# TC_PUMB_002: Вибір способу "Без участі менеджера"

## Основна інформація
- **ID:** TC_PUMB_002
- **Title:** Перевірка вибору онлайн реєстрації без участі менеджера
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 4m
- **Automated:** YES
- **References:** US_PUMB_002

## Preconditions
- Сторінка реєстрації ФОП завантажена
- Відображаються два варіанти реєстрації

## Steps
1. Знайти секцію "Без участі менеджера"
2. Перевірити відображення підзаголовка "Онлайн"
3. Перевірити список вимог для онлайн реєстрації
4. Натиснути на кнопку/посилання для переходу до онлайн реєстрації

## Expected Result
Відображається секція "Без участі менеджера" з підзаголовком "Онлайн". Показані вимоги: "Застосунок Дія з ID-карткою або закордонним паспортом" та "Застосунок ПУМБ для фізичних осіб". Кнопка активна та дозволяє перейти до наступного кроку.

## Postconditions
Повернутися на головну сторінку реєстрації

# TC_PUMB_010: Адаптивність сторінки на різних розмірах екрану

## Основна інформація
- **ID:** TC_PUMB_010
- **Title:** Перевірка адаптивності сторінки на різних розмірах екрану
- **Type:** UI
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 5m
- **Automated:** YES
- **References:** US_PUMB_010

## Preconditions
- Сторінка реєстрації ФОП доступна
- Браузер підтримує зміну розміру вікна

## Steps
1. Відкрити сторінку на десктопі (1920x1080)
2. Змінити розмір вікна на планшет (768x1024)
3. Змінити розмір вікна на мобільний (375x667)
4. Перевірити відображення контенту на кожному розмірі

## Expected Result
Сторінка коректно відображається на всіх розмірах екрану. На мобільних пристроях контент адаптується, зберігаючи читабельність. Всі елементи інтерфейсу залишаються доступними та функціональними.

## Postconditions
Повернути оригінальний розмір вікна браузера

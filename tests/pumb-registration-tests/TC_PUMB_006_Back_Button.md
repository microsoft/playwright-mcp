# TC_PUMB_006: Клік по кнопці "Повернутись"

## Основна інформація
- **ID:** TC_PUMB_006
- **Title:** Перевірка функціональності кнопки "Повернутись"
- **Type:** Functional
- **Priority:** Low
- **Status Case:** Positive
- **Estimate:** 2m
- **Automated:** YES
- **References:** US_PUMB_006

## Preconditions
- Сторінка реєстрації ФОП завантажена
- Кнопка "Повернутись" присутня на сторінці

## Steps
1. Знайти кнопку "Повернутись" на сторінці
2. Перевірити, що кнопка видима та активна
3. Натиснути на кнопку "Повернутись"
4. Перевірити результат натискання

## Expected Result
Кнопка "Повернутись" відображається на сторінці та є клікабельною. При натисканні відбувається перехід на попередню сторінку або головну сторінку сайту ПУМБ.

## Postconditions
Повернутися на сторінку реєстрації ФОП

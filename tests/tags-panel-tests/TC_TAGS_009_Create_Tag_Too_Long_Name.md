# TC_TAGS_009: Створення тегу з занадто довгою назвою

## Основна інформація
- **ID:** TC_TAGS_009
- **Title:** Спроба створення тегу з назвою, що перевищує ліміт символів
- **Type:** Validation
- **Priority:** Medium
- **Status Case:** Negative
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_009

## Preconditions
- Користувач авторизований у системі
- Відкрита форма створення тегу
- Максимальна довжина назви тегу: 50 символів

## Steps
1. Натиснути кнопку "Додати тег"
2. Ввести назву тегу довжиною 51 символ
3. Натиснути "Зберегти"

## Expected Result
Відображається повідомлення про помилку "Назва тегу не може перевищувати 50 символів". Тег не створюється.

## Postconditions
Закрити форму створення тегу

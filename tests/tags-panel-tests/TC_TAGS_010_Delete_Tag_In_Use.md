# TC_TAGS_010: Видалення тегу, який використовується в картках

## Основна інформація
- **ID:** TC_TAGS_010
- **Title:** Спроба видалення тегу, що прив'язаний до карток
- **Type:** Functional
- **Priority:** Medium
- **Status Case:** Negative
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_010

## Preconditions
- Користувач авторизований у системі
- Існує тег, який використовується принаймні в одній картці

## Steps
1. В панелі тегів знайти тег, що використовується
2. Натиснути на іконку "Видалити" біля тегу
3. Підтвердити видалення

## Expected Result
Відображається попередження "Цей тег використовується в X картках. Ви впевнені, що хочете його видалити?" з опціями "Так" та "Ні".

## Postconditions
Відмінити видалення, натиснувши "Ні"

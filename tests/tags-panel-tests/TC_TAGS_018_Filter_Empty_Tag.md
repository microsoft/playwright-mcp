# TC_TAGS_018: Фільтрація при відсутності карток з тегом

## Основна інформація
- **ID:** TC_TAGS_018
- **Title:** Фільтрація за тегом, який не має прив'язаних карток
- **Type:** Functional
- **Priority:** Low
- **Status Case:** Positive
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_018

## Preconditions
- Користувач авторизований у системі
- Існує тег без прив'язаних карток
- Відкрита головна сторінка

## Steps
1. В панелі тегів знайти тег без карток
2. Натиснути на цей тег для фільтрації

## Expected Result
Відображається повідомлення "Картки з цим тегом не знайдено" або порожній список карток.

## Postconditions
Скинути фільтр

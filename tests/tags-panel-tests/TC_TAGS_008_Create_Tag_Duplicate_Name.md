# TC_TAGS_008: Створення тегу з дублікатом назви

## Основна інформація
- **ID:** TC_TAGS_008
- **Title:** Спроба створення тегу з назвою, що вже існує
- **Type:** Validation
- **Priority:** High
- **Status Case:** Negative
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_008

## Preconditions
- Користувач авторизований у системі
- Існує тег з назвою "Математика"
- Відкрита форма створення тегу

## Steps
1. Натиснути кнопку "Додати тег"
2. Ввести назву "Математика" (існуючий тег)
3. Натиснути "Зберегти"

## Expected Result
Відображається повідомлення про помилку "Тег з такою назвою вже існує". Тег не створюється.

## Postconditions
Закрити форму створення тегу

- **Title:** Пошук тегів за назвою в панелі
- **Type:** Functional
- **Priority:** Medium

- **Automated:** NO

## Preconditions
- Користувач авторизований у системі

## Steps
1. В полі пошуку тегів ввести частину назви тегу
2. Перевірити результати пошуку

## Expected Result
Відображаються тільки теги, назви яких містять введений текст. Пошук працює в реальному часі.

# TC_TAGS_011: Відображення панелі тегів

## Основна інформація
- **ID:** TC_TAGS_011
- **Title:** Перевірка коректного відображення панелі тегів
- **Type:** UI
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_011

## Preconditions
- Користувач авторизований у системі
- Відкрита головна сторінка додатку

## Steps
1. Перевірити наявність панелі тегів на сторінці
2. Перевірити відображення існуючих тегів
3. Перевірити наявність кнопки додавання тегу

## Expected Result
Панель тегів відображається в правильному місці. Всі існуючі теги показані з правильними назвами. Кнопка додавання тегу видима та доступна.

## Postconditions
Не потрібно

# TC_TAGS_013: Візуальний стан активного тегу

## Основна інформація
- **ID:** TC_TAGS_013
- **Title:** Перевірка виділення активного тегу при фільтрації
- **Type:** UI
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_013

## Preconditions
- Користувач авторизований у системі
- Існують картки з різними тегами
- Відкрита головна сторінка

## Steps
1. Натиснути на тег для фільтрації
2. Перевірити візуальне виділення активного тегу
3. Натиснути на інший тег
4. Перевірити зміну виділення

## Expected Result
Активний тег візуально виділяється (зміна кольору, підсвічування, рамка). При виборі іншого тегу виділення переходить на новий тег.

## Postconditions
Скинути фільтр

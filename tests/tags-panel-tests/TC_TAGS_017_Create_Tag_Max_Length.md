# TC_TAGS_017: Створення тегу з максимальною довжиною

## Основна інформація
- **ID:** TC_TAGS_017
- **Title:** Створення тегу з назвою максимальної довжини
- **Type:** Functional
- **Priority:** Low
- **Status Case:** Positive
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_017

## Preconditions
- Користувач авторизований у системі
- Відкрита форма створення тегу
- Максимальна довжина назви тегу: 50 символів

## Steps
1. Натиснути кнопку "Додати тег"
2. Ввести назву довжиною рівно 50 символів
3. Натиснути "Зберегти"

## Expected Result
Тег успішно створений з назвою максимальної довжини і відображається в панелі тегів.

## Postconditions
Видалити створений тестовий тег

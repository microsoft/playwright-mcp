# TC_TAGS_015: Валідація пошуку тегів

## Основна інформація
- **ID:** TC_TAGS_015
- **Title:** Перевірка пошуку тегів з порожнім запитом
- **Type:** Validation
- **Priority:** Low
- **Status Case:** Negative
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_015

## Preconditions
- Користувач авторизований у системі
- Існують теги в системі
- Відкрита панель тегів

## Steps
1. В поле пошуку ввести пробіли
2. Натиснути Enter або кнопку пошуку
3. Перевірити результати

## Expected Result
Порожній запит ігнорується, відображаються всі теги. Або відображається повідомлення "Введіть назву тегу для пошуку".

## Postconditions
Очистити поле пошуку

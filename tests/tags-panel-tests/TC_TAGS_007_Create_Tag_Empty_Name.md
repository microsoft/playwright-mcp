# TC_TAGS_007: Створення тегу з порожньою назвою

## Основна інформація
- **ID:** TC_TAGS_007
- **Title:** Спроба створення тегу без назви
- **Type:** Validation
- **Priority:** High
- **Status Case:** Negative
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_007

## Preconditions
- Користувач авторизований у системі
- Відкрита форма створення тегу

## Steps
1. Натиснути кнопку "Додати тег"
2. Залишити поле назви порожнім
3. Натиснути "Зберегти"

## Expected Result
Відображається повідомлення про помилку "Назва тегу не може бути порожньою". Тег не створюється.

## Postconditions
Закрити форму створення тегу

# TC_TAGS_002: Додавання тегу до картки

## Основна інформація
- **ID:** TC_TAGS_002
- **Title:** Додавання існуючого тегу до flashcard
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 5m
- **Automated:** NO
- **References:** US_TAGS_002

## Preconditions
- Користувач авторизований у системі
- Існує принаймні один тег у системі
- Відкрита форма створення/редагування картки

## Steps
1. Відкрити форму створення нової картки
2. Заповнити поля "Питання" та "Відповідь"
3. В полі "Теги" почати вводити назву існуючого тегу
4. Вибрати тег з випадаючого списку
5. Натиснути "Зберегти картку"

## Expected Result
Картка успішно створена з доданим тегом. Тег відображається у картці і картка з'являється при фільтрації за цим тегом.

## Postconditions
Видалити створену тестову картку

# TC_TAGS_020: Пошук серед великої кількості тегів

## Основна інформація
- **ID:** TC_TAGS_020
- **Title:** Перевірка швидкості пошуку серед 100+ тегів
- **Type:** Other
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_020

## Preconditions
- Користувач авторизований у системі
- В системі створено 100+ тегів
- Відкрита панель тегів

## Steps
1. В поле пошуку ввести частину назви тегу
2. Заміряти час відображення результатів
3. Перевірити точність пошуку

## Expected Result
Результати пошуку відображаються за час не більше 1 секунди. Показуються тільки релевантні теги.

## Postconditions
Очистити поле пошуку

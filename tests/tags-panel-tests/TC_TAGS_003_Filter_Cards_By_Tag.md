# TC_TAGS_003: Фільтрація карток за тегом

## Основна інформація
- **ID:** TC_TAGS_003
- **Title:** Відображення карток за обраним тегом
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_003

## Preconditions
- Користувач авторизований у системі
- Існують картки з різними тегами
- Відкрита головна сторінка зі списком карток

## Steps
1. В панелі тегів знайти потрібний тег
2. Натиснути на тег для фільтрації
3. Перевірити відображення карток

## Expected Result
Відображаються тільки картки, які мають обраний тег. Кількість карток відповідає реальній кількості карток з цим тегом.

## Postconditions
Скинути фільтр, натиснувши "Показати всі" або "Очистити фільтри"

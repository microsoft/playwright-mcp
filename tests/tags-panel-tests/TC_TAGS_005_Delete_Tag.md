# TC_TAGS_005: Видалення тегу

## Основна інформація
- **ID:** TC_TAGS_005
- **Title:** Видалення тегу з системи
- **Type:** Functional
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_005

## Preconditions
- Користувач авторизований у системі
- Існує тег, який потрібно видалити
- Користувач має права на видалення тегів

## Steps
1. В панелі тегів знайти тег для видалення
2. Натиснути на іконку "Видалити" біля тегу
3. Підтвердити видалення в діалоговому вікні

## Expected Result
Тег успішно видалений з системи. Тег більше не відображається в панелі тегів та видаляється з усіх карток, які його використовували.

## Postconditions
Не потрібно (тег видалений)

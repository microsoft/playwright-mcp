# TC_TAGS_012: Адаптивність панелі тегів

## Основна інформація
- **ID:** TC_TAGS_012
- **Title:** Перевірка відображення панелі тегів на різних розмірах екрану
- **Type:** UI
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 5m
- **Automated:** NO
- **References:** US_TAGS_012

## Preconditions
- Користувач авторизований у системі
- Відкрита головна сторінка додатку

## Steps
1. Відкрити сайт на десктопі (1920x1080)
2. Змінити розмір вікна на планшет (768x1024)
3. Змінити розмір вікна на мобільний (375x667)

## Expected Result
Панель тегів коректно відображається на всіх розмірах екрану. На мобільних пристроях теги можуть відображатися в вигляді випадаючого меню або згортатися.

## Postconditions
Повернути оригінальний розмір вікна

# TC_TAGS_004: Редагування назви тегу

## Основна інформація
- **ID:** TC_TAGS_004
- **Title:** Зміна назви існуючого тегу
- **Type:** Functional
- **Priority:** Medium
- **Status Case:** Positive
- **Estimate:** 4m
- **Automated:** NO
- **References:** US_TAGS_004

## Preconditions
- Користувач авторизований у системі
- Існує тег, який потрібно редагувати
- Користувач має права на редагування тегів

## Steps
1. В панелі тегів знайти тег для редагування
2. Натиснути на іконку "Редагувати" біля тегу
3. Змінити назву тегу
4. Натиснути "Зберегти"

## Expected Result
Назва тегу успішно змінена. Нова назва відображається в панелі тегів та у всіх картках, які використовують цей тег.

## Postconditions
Повернути оригінальну назву тегу

# TC_TAGS_016: Створення тегу з мінімальною довжиною

## Основна інформація
- **ID:** TC_TAGS_016
- **Title:** Створення тегу з назвою в 1 символ
- **Type:** Functional
- **Priority:** Low
- **Status Case:** Positive
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_016

## Preconditions
- Користувач авторизований у системі
- Відкрита форма створення тегу
- Мінімальна довжина назви тегу: 1 символ

## Steps
1. Натиснути кнопку "Додати тег"
2. Ввести один символ "А"
3. Натиснути "Зберегти"

## Expected Result
Тег успішно створений з назвою "А" і відображається в панелі тегів.

## Postconditions
Видалити створений тестовий тег

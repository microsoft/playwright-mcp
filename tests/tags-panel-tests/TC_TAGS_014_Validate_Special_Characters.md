# TC_TAGS_014: Валідація спеціальних символів у назві тегу

## Основна інформація
- **ID:** TC_TAGS_014
- **Title:** Перевірка обробки спеціальних символів при створенні тегу
- **Type:** Validation
- **Priority:** Medium
- **Status Case:** Negative
- **Estimate:** 2m
- **Automated:** NO
- **References:** US_TAGS_014

## Preconditions
- Користувач авторизований у системі
- Відкрита форма створення тегу

## Steps
1. Натиснути кнопку "Додати тег"
2. Ввести назву з спеціальними символами: "Тест@#$%"
3. Натиснути "Зберегти"

## Expected Result
Відображається повідомлення про помилку "Назва тегу може містити тільки літери, цифри та пробіли". Тег не створюється.

## Postconditions
Закрити форму створення тегу

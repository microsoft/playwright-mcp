# TC_TAGS_001: Створення нового тегу

## Основна інформація
- **ID:** TC_TAGS_001
- **Title:** Створення нового тегу через панель тегів
- **Type:** Functional
- **Priority:** High
- **Status Case:** Positive
- **Estimate:** 3m
- **Automated:** NO
- **References:** US_TAGS_001

## Preconditions
- Користувач авторизований у системі
- Відкрита панель тегів
- Користувач має права на створення тегів

## Steps
1. Натиснути кнопку "Додати тег" або "+" в панелі тегів
2. Ввести назву нового тегу (наприклад, "Математика")
3. Натиснути кнопку "Зберегти" або "Enter"

## Expected Result
Новий тег успішно створений і відображається в списку тегів. Тег доступний для використання при створенні/редагуванні карток.

## Postconditions
Видалити створений тестовий тег з системи

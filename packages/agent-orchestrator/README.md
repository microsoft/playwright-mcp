# Agent Orchestrator & Routing Manager

This package implements the central orchestrator for the multi-agent AI system. Its primary responsibility is to decompose complex user requests into subtasks and route them to the appropriate specialist agents.

## Specification
- **Version**: 1.0
- **Status**: Implementation Phase
- **Target Repo**: `msitarzewskiagency-agents`

## Key Responsibilities
1. **Request Analysis**: Classify user intent and determine the required specialists.
2. **Task Decomposition**: Break down high-level goals into executable steps.
3. **Agent Selection**: Match subtasks to agents.
4. **State Management**: Maintain task state and audit logs.
